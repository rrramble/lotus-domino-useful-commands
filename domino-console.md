## Exit console
- ```quit```
- ```exit```

## Restart Domino server
```restart server```

## Run a task
- ```load <TASK_NAME>```
- ```load <EXE_FILE> <EXE_FILE_PARAMTERS>```

Example:
```load cmd /c c:\windows\notepad.exe```

## Manage a task
```tell <TASK_NAME> <PARAMETER>```

PARAMETER can be:
- ```quit```
- ```restart```
